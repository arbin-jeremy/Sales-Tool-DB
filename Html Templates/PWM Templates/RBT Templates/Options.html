Options
Arbin can add most of the auxiliary options listed to an existing system. The new auxiliary modules are provided in a small stand-alone box that is plug and play with the existing Arbin tester.
Arbin system also has various optional functions, accessories and advanced software packages available.
Optional Function Description
EIS Interface* Allows connection of Arbin EIS modules or select Gamry EIS modules that are multiplexed across all channels. Channels automatically queue with simultaneous EIS requests. No change in connection is required. EIS device is sold separately.
Optional Built-in Auxiliary* Some Arbin IV channels can add built-in auxiliary channels, including Temperature/Voltage reading.
Smart Battery SMBus （built-in） SMBus communication; Allows Arbin system to send and receive signals using SMBus protocol with the battery under test.
Compatible with SMBus 2.0 & 1.1 protocols, and tested with all common gas gauge IC’s from Texas Instruments (TI).
Read & Write SMBus Registers, and use them to control or limit the test.
Each SMBus input will allow the independent control of a smart battery.
Data is stored in an SQL database and can be automatically imported into Excel for each viewing and analysis.
Data may be viewed in real-time, including SMBus register values while test is running.
Auxiliaries Description
Auxiliary Voltage Various voltage ranges for monitoring cell voltages, or for use as additional reference electrodes.
Auxiliary Temperature Inputs for thermocouples or thermistor to measure and record temperature.
Temperature can be used as a safety limit and used to control the test.
All temperature inputs are independent and can be mapped by the user with the main I/V channels in any configuration. 1:1, 1:many, many:1, and anything in between.
NOTE: Arbin offers Type-T or K thermocouple, or PT100 or 10kOhm thermistor inputs for temperature measurement.
Auxiliary Pressure The Pressure Input option can be used to record pressure measurements with pressure transducers and works in a similar way as the 2nd voltage measurement.
Analog Input/Output Relay Sends and receives analog signals, this auxiliary module can interface with any devices that can output the same feedback signal and input the same control signal. The Analog I/O option offers control with closed loop (PID) or open loop available according to the application.
Digital Input/Output Relay Send and receive a simple digital on/off signal.
Temperature Chamber Interface(MTCI) Allows interface to an approved 3rd party temperature chamber.
CANBus CAN Bus Communication. Allows Arbin system to receive and send CAN signals. No 3rd party software or licenses are required. The Arbin system can be controlled using CAN protocols from the BMS.
Arbin RBT will receive and write data to the BMS without needing external licenses or DLL packages.
Arbin RBT is the only system that provides full control of the tester using CANBus protocol from BMS or other external source. i.e. Charge based on “CAN_variable_01” instead of a numeric value.
Arbin RBT is the only system offering a unique power clamp that uses a numeric value or CANBus variable that can be dynamically changed during the test.
This will limit power output of tests (especially simulations) while running without needing to stop, edit, and start the test over again.
Arbin EIS Perform Galvanostatic EIS tests for devices under test.
AutoCal "AutoCal" module to automate the calibration process. Software interface for manual calibration is a standard feature for all Arbin testers, but AutoCal module is recommended with more than 32 test channels.
Arbin MZTC Multi-Chamber The MZTC Multi-Chamber provides 8 fully independent mini-chambers that safely isolate cells and control temperature between 10-60 °C.
Accessories Description
Cables Various cables available. For Current or signals, equipped with different length and terminals with alligator clips, lugs, banana plugs, or bare leads.
Battery Holders Coin cell, cylindrical cell, and flat cell holders available.
Battery Trays and Rack Various designs of battery trays and rack for cell holders available.
UPS 1500Wh UPS provides backup power to PC only and allows tests to resume automatically if power is restored quickly.
